
=== COMO USAR ===

1) Suba este projeto no GitHub.
2) Conecte na Netlify via 'Import from Git'.
3) Vá em Data & Storage → Netlify DB → Create database.
4) Netlify criará automaticamente a variável NETLIFY_DATABASE_URL.
5) Em Site settings → Access control → coloque um SITE PASSWORD
   (isso protege o site inteiro).
6) Redeploy.

Pronto. Sem login interno no app.
